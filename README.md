# Image Convert Resize and Crop
----------

*** Description ***
-----

An image tool class for :

 - Converting
 - Resizing
 - Cropping
 - Percent and dimensions mode
 - Keep ratio
 - PNG, JPEG, GIF support
 - JPEG quality (0-100)
 - PNG compression (1-9)
 - Native shell mode vs GD (default)

*** How to use ***
-----

Look at index.php
